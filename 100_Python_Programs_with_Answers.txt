6. Find Factorial
# Program 6: Factorial
def factorial(n):
    if n == 0 or n == 1:
        return 1
    return n * factorial(n - 1)

n = 4
print("Factorial is:", factorial(n))
Output:
Factorial is: 24
----------------------------------------

8. Reverse a Number
# Program 8: Reverse a Number
n = 1234
rev = 0
while n > 0:
    rev = rev * 10 + n % 10
    n = n // 10
print("Reversed number:", rev)
Output:
Reversed number: 4321
